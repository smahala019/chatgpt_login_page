# 📦 Universal Auth UI Simulation

A collection of high-fidelity, responsive authentication UI screens built using Vanilla HTML, CSS, and JavaScript.

This project simulates login and account recovery flows inspired by platforms like Google, Apple, Microsoft, and Facebook.

## 🛠️ Tech Stack / Languages Used

This project is built using only core web technologies:

### 💻 Frontend
- HTML5 → Page structure and UI layout
- CSS3 → Styling, responsive design, animations
- JavaScript (ES6+) → Logic, validation, CAPTCHA, OTP, i18n

### ⚙️ Browser APIs Used
- Canvas API → CAPTCHA generation with noise and distortion
- DOM API → Dynamic UI updates and navigation
- Timer APIs → setTimeout, setInterval for OTP countdown


# ⚠️ Disclaimer

This project is for educational and UI/UX demonstration purposes only.

No real authentication system is implemented

No backend or database is used

OTP and login flows are simulated in the browser

UI design is inspired by real platforms for learning purposes only
